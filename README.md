# Git 打卡教程

## Fork 主仓库

- 注册/登陆 GitHub
- 在本仓库页面，点击右上角 `Fork` 按扭

## 填写打卡信息

在 fork 到个人账号的项目中根目录提交 `<名称>.toml` 及头像图片文件: 

```toml
[about]
name = "djdjz"
avatar = "djdjz.jpeg"
hobbies = ["🎨 Desgin", "💻 Programming", "🏸 Badminton"]
skills = ["C#", "Rust", "TypeScript", "Vue"]

[social]
github = "djdjz7"
website = "https://djdjz7.top"
```

## 发起 PR

完成提交后，返回主仓库页面，根据界面上的提示将提交的信息 Pr 到主仓库。
